# BudgetFlow API Reference (Phase 1)

This document lists all Phase 1 API endpoints. Full OpenAPI/Swagger is available at `GET /api/v1/openapi.json` when the backend is running.

## Base URL

- Local: `http://localhost:8000`
- All routes under `/api/v1` unless noted.

## Authentication

- **Signup / Login:** Obtain `access_token` and optionally `refresh_token`.
- **Authenticated requests:** Header: `Authorization: Bearer <access_token>`.
- **Refresh:** `POST /api/v1/auth/refresh` with body `{ "refresh_token": "..." }` to get a new access token.

## Error Response Format

All errors return JSON with a user-facing message:

```json
{
  "detail": "Human-readable message",
  "code": "OPTIONAL_ERROR_CODE",
  "errors": []
}
```

Common status codes: `400` Bad Request, `401` Unauthorized, `403` Forbidden, `404` Not Found, `422` Unprocessable Entity (e.g. validation).
